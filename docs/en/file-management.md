## Basic Operations

### Upload Files

#### Method 1: Drag & Drop
1. Open your computer's file folder
2. Select files or folders to upload
3. Drag them to the browser window's file area
4. Release mouse and wait for upload to complete

#### Method 2: Click Upload
1. Click the "Upload" button in the top toolbar
2. Select files in the popup window
3. Click "Open" to start upload

> üí° **Tip**: When uploading large files, keep the page open until upload completes. You can select multiple files for batch upload.

### Create Folder

1. Click the "New Folder" button in the top toolbar
2. Enter folder name (e.g., "Work Documents", "Family Photos")
3. Press Enter or click confirm

### Select Files

- **Single Select**: Click on filename
- **Multi Select**: Hold Ctrl (Command on Mac) and click multiple files
- **Select All**: Click the checkbox at the top of the list
- **Range Select**: Select one file, hold Shift and select another to select all files in between

### Move and Copy

#### Move Files
1. Select files to move
2. Click the "Move to" button
3. Select target folder in the popup window
4. Click confirm

#### Copy Files
1. Select files to copy
2. Click the "Copy to" button
3. Select target location
4. Click confirm

> üí° **Tip**: You can also drag and drop files directly to the target folder.

### Rename

1. Right-click on file or folder
2. Select "Rename"
3. Enter new name
4. Press Enter to confirm

### Delete Files

1. Select files to delete
2. Click the "Delete" button or press Delete key
3. Click "OK" in the confirmation dialog

> ‚ö†Ô∏è **Note**: Deleted files go to the recycle bin and can be restored within 30 days.

## View Modes

### List View
- Shows file details (name, size, modification time)
- Suitable for viewing large numbers of files
- Click column headers to sort

### Icon View
- Large icon display
- Image files show thumbnails
- Suitable for browsing image folders

Switch between views by clicking the view toggle button in the top right.

## Quick Navigation

### Address Bar
The address bar shows your current location. You can:
- Click folder names to jump to that level
- Click dropdown arrows to see all folders at that level

### Left Sidebar
- **My**: Main entry to all your files
- **Favorites**: Files and folders you've marked as favorites
- **Shares**: All share links you've created
- **Recycle Bin**: Deleted files

### Favorites
Add frequently used folders to favorites for quick access:
1. Navigate to the folder you want to favorite
2. Click the "Favorite" button (star icon) in the toolbar
3. Access it directly from the "Favorites" section in the sidebar

## Search Function

### Simple Search
1. Click the search box at the top
2. Enter filename or keywords
3. Press Enter, results will display in real-time

### Search Tips
- Supports fuzzy search, just enter partial names
- Search results highlight matching parts
- Search scope includes current folder and subfolders

## File Preview

### Supported Preview Types
- **Images**: JPG, PNG, GIF, BMP, etc.
- **Documents**: PDF, TXT
- **Videos**: MP4, WebM and other mainstream formats
- **Audio**: MP3, WAV, etc.

### How to Preview
- Click on image or video files to open preview directly
- Double-click files to open in new tab
- Right-click and select "Preview"

### Preview Operations
- Images: Arrow keys to switch, zoom in/out, download
- Videos: Play/pause, drag progress bar, fullscreen
- Documents: Page turn, zoom, download

## Batch Operations

### Batch Upload
Drag multiple files or entire folders to the upload area, the system will automatically process all files.

### Batch Download
1. Select multiple files
2. Click the "Download" button
3. The system will package them into a ZIP file for download

### Batch Delete
1. Select multiple files
2. Click the "Delete" button
3. Confirm to delete in batch

## FAQ

**Q: What if the upload disconnects for large files?**
A: Re-upload the same file, the system will automatically resume from the break point.

**Q: Why can't some files be previewed?**
A: Some special formats may need to be downloaded and opened with professional software. See supported preview formats above.

**Q: How to view folder size?**
A: In list view, folders will display the total size of their contents.

**Q: What if I accidentally deleted a file?**
A: Click "Recycle Bin" in the left sidebar, find the deleted file, and click the "Restore" button.
