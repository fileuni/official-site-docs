# FileUni 产品介绍

## 什么是 FileUni？

FileUni 是一款**私人文件管理系统**，专为希望在自有设备上搭建个人云存储的用户设计。

想象您有一个智能文件柜，它可以：
- 📁 **集中存储**：把所有文件放在一个安全的地方
- 🌐 **随时访问**：通过电脑、手机、平板随时访问您的文件
- 👥 **多人共享**：与家人、朋友或同事安全地分享文件
- 🔒 **隐私保护**：数据存在您自己的设备上，而不是大公司的服务器

## 适合谁使用？

| 使用场景 | 说明 |
|---------|------|
| **家庭用户** | 备份照片、视频，建立家庭媒体中心 |
| **小型团队** | 共享工作文档，协作办公 |
| **个人用户** | 统一管理分散在各个设备的文件 |
| **老旧设备利用** | 让旧电脑、旧手机变身文件服务器 |

## 核心优势

### 🚀 轻量快速
- 即使老旧设备也能流畅运行
- 最低只需 512MB 内存
- 启动速度快，占用资源少

### 🔌 多种访问方式
- **网页浏览器**：像使用网盘一样简单
- **手机 APP**：支持标准协议，可用第三方客户端
- **电脑同步**：像访问本地硬盘一样方便

### 🛡️ 数据安全
- 您的文件存储在自己的设备上
- 多层防护防止未授权访问
- 误删文件可一键恢复

### 📱 全平台支持
- Windows、Mac、Linux 电脑
- 安卓和苹果手机
- 路由器、树莓派等设备

## 与传统网盘的区别

| 功能 | 传统网盘 | FileUni |
|------|---------|---------|
| 数据归属 | 存储在服务商服务器 | 存储在您自己的设备 |
| 隐私控制 | 服务商可能查看文件 | 完全由您掌控 |
| 容量限制 | 需要付费扩容 | 取决于您的硬盘大小 |
| 访问速度 | 受网络环境影响 | 局域网内极速访问 |
| 月费 | 通常需要订阅 | 一次部署，永久使用 |

## 使用场景示例

### 📸 照片备份
把手机里的照片自动备份到家里的设备，释放手机空间，同时保留珍贵回忆。

### 🎬 家庭影院
将电影、音乐存储在 FileUni 上，全家在任何设备上都能观看。

### 📂 工作协作
小团队共享项目文件，每个人都能看到最新版本，告别文件传来传去。

### 💾 资料归档
将不常用但又不想删除的文件归档存储，需要时随时调取。
