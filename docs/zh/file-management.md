# 文件管理指南

RS-Core 的文件管理界面设计简洁直观，就像使用电脑上的文件夹一样简单。

## 界面布局

打开网页界面后，您会看到三个主要区域：

```
┌─────────────────────────────────────────────────────┐
│  顶部工具栏 (上传、新建、搜索等操作)                    │
├──────────┬──────────────────────────────────────────┤
│          │                                          │
│  左侧    │              主文件区域                   │
│  导航栏  │          (文件列表/图标视图)              │
│          │                                          │
│  - 我的  │                                          │
│  - 收藏  │                                          │
│  - 分享  │                                          │
│  - 回收站│                                          │
│          │                                          │
└──────────┴──────────────────────────────────────────┘
```

## 基础操作

### 上传文件

#### 方法一：拖拽上传
1. 打开电脑的文件夹
2. 选中要上传的文件或文件夹
3. 拖拽到浏览器窗口的文件区域
4. 松开鼠标，等待上传完成

#### 方法二：点击上传
1. 点击顶部工具栏的「上传」按钮
2. 在弹出的窗口中选择文件
3. 点击「打开」开始上传

> 💡 **小贴士**：上传大文件时，请保持页面打开直到上传完成。可以一次选择多个文件批量上传。

### 创建文件夹

1. 点击顶部工具栏的「新建文件夹」按钮
2. 输入文件夹名称（如："工作文档"、"家庭照片"）
3. 按回车键或点击确认

### 选择文件

- **单选**：点击文件名
- **多选**：按住 Ctrl（Mac 为 Command）键，依次点击多个文件
- **全选**：点击列表顶部的复选框
- **范围选择**：先选一个文件，按住 Shift 再选另一个，中间的所有文件都会被选中

### 移动和复制

#### 移动文件
1. 选中要移动的文件
2. 点击「移动到」按钮
3. 在弹出的窗口中选择目标文件夹
4. 点击确认

#### 复制文件
1. 选中要复制的文件
2. 点击「复制到」按钮
3. 选择目标位置
4. 点击确认

> 💡 **小贴士**：也可以使用拖拽方式，直接拖动文件到目标文件夹。

### 重命名

1. 右键点击文件或文件夹
2. 选择「重命名」
3. 输入新名称
4. 按回车确认

### 删除文件

1. 选中要删除的文件
2. 点击「删除」按钮，或按键盘上的 Delete 键
3. 在弹出的确认框中点击「确定」

> ⚠️ **注意**：删除的文件会进入回收站，30 天内可以从回收站恢复。

## 查看模式

### 列表视图
- 显示文件详细信息（名称、大小、修改时间）
- 适合查看大量文件
- 点击列标题可以排序

### 图标视图
- 大图标显示
- 图片文件显示缩略图
- 适合浏览图片文件夹

切换方式：点击右上角的视图切换按钮

## 快速导航

### 地址栏
地址栏显示当前所在位置，点击可以：
- 点击文件夹名称跳转到对应层级
- 点击下拉箭头查看该层级的所有文件夹

### 左侧导航
- **我的**：所有文件的总入口
- **收藏**：您标记为收藏的文件和文件夹
- **分享**：您创建的所有分享链接
- **回收站**：已删除的文件

### 收藏功能
把常用文件夹添加到收藏，方便快速访问：
1. 进入要收藏的文件夹
2. 点击工具栏的「收藏」按钮（星形图标）
3. 以后可以从左侧「收藏」直接访问

## 搜索功能

### 简单搜索
1. 点击顶部的搜索框
2. 输入文件名或关键词
3. 按回车，系统会实时显示结果

### 搜索技巧
- 支持模糊搜索，输入部分名称即可
- 搜索结果会高亮显示匹配的部分
- 搜索范围是当前文件夹及其子文件夹

## 文件预览

### 支持的预览类型
- **图片**：JPG、PNG、GIF、BMP 等
- **文档**：PDF、TXT
- **视频**：MP4、WebM 等主流格式
- **音频**：MP3、WAV 等

### 如何预览
- 单击图片或视频文件直接打开预览
- 双击文件在新标签页打开
- 右键选择「预览」

### 预览操作
- 图片：左右箭头切换、放大缩小、下载
- 视频：播放/暂停、进度条拖动、全屏
- 文档：翻页、缩放、下载

## 批量操作

### 批量上传
拖拽多个文件或整个文件夹到上传区域，系统会自动处理所有文件。

### 批量下载
1. 选中多个文件
2. 点击「下载」按钮
3. 系统会打包成 ZIP 文件下载

### 批量删除
1. 选中多个文件
2. 点击「删除」按钮
3. 确认后批量删除

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl + A | 全选 |
| Ctrl + C | 复制选中文件 |
| Ctrl + V | 粘贴 |
| Delete | 删除选中文件 |
| F2 | 重命名 |
| Ctrl + F | 搜索 |

## 常见问题

**Q: 上传大文件时断开了怎么办？**
A: 重新上传同一文件，系统会自动从断点续传。

**Q: 为什么有些文件无法预览？**
A: 某些特殊格式可能需要下载后用专业软件打开。支持的预览格式请参考上文。

**Q: 如何查看文件夹大小？**
A: 在列表视图中，文件夹会显示其包含内容的总大小。

**Q: 误删了文件怎么办？**
A: 点击左侧「回收站」，找到删除的文件，点击「恢复」按钮。
