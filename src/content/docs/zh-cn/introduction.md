# FileUni 产品介绍

## 什么是 FileUni？

FileUni 是一款**私人文件管理系统**，专为希望在自有设备上搭建个人云存储的用户设计。

想象您有一个智能文件柜，它可以：
- 📁 **集中存储**：把所有文件放在一个安全的地方
- 🌐 **随时访问**：通过电脑、手机、平板随时访问您的文件
- 👥 **多人共享**：与家人、朋友或同事安全地分享文件
- 🔒 **隐私保护**：数据存在您自己的设备上，而不是大公司的服务器

## 适合谁使用？

| 使用场景 | 说明 |
|---------|------|
| **家庭用户** | 备份照片、视频，建立家庭媒体中心 |
| **小型团队** | 共享工作文档，协作办公 |
| **个人用户** | 统一管理分散在各个设备的文件 |
| **老旧设备利用** | 让旧电脑、旧手机变身文件服务器 |

## 核心优势

### 🚀 极致轻量与弹性扩展
- **极低资源需求**：最低 **32MB 内存** 即可运行，64MB 内存可流畅运行
- **自适应性能**：从路由器到高性能服务器，资源越多性能越好
- **无需重新分区**：直接在现有系统上安装，无需重新格式化硬盘
- **轻松迁移**：重装或升级系统时，数据结构完整保留

### 🔌 多种访问方式
- **网页浏览器**：像使用网盘一样简单
- **手机 APP**：支持标准协议，可用第三方客户端
- **电脑同步**：像访问本地硬盘一样方便

### 🛡️ 数据安全
- 您的文件存储在自己的设备上
- 多层防护防止未授权访问
- 误删文件可一键恢复

### 📱 全平台支持
- Windows、Mac、Linux 电脑
- 安卓和苹果手机
- 路由器、树莓派、NAS 设备，甚至老旧硬件

## 与传统网盘的区别

| 功能 | 传统网盘 | FileUni |
|------|---------|---------|
| 数据归属 | 存储在服务商服务器 | 存储在您自己的设备 |
| 隐私控制 | 服务商可能查看文件 | 完全由您掌控 |
| 容量限制 | 需要付费扩容 | 取决于您的硬盘大小 |
| 访问速度 | 受网络环境影响 | 局域网内极速访问 |
| 月费 | 通常需要订阅 | 一次部署，永久使用 |
| 硬件要求 | 无 | 低至 32MB 内存即可运行 |

## 真正的 NAS 替代方案

FileUni 以极低的资源开销，提供主流 NAS 的大部分功能：

### 💡 为什么选择 FileUni 而不是传统 NAS？

| 方面 | 传统 NAS | FileUni |
|------|---------|---------|
| **硬件成本** | 专用 NAS 设备（200 美元以上） | 任何您现有的设备 |
| **内存需求** | 通常 2GB 以上 | **低至 32MB** |
| **设置复杂度** | RAID 配置、分区规划 | 安装即用 |
| **系统迁移** | 复杂的备份恢复 | 无需重新分区，无缝升级 |
| **功耗** | 专用硬件 15-50W | 运行在现有设备上 |

### 🔄 无忧系统迁移

这是 FileUni 的一大亮点：

- **无需重新分区**：在现有系统上直接安装，无需重新格式化硬盘
- **保留数据结构**：重装或升级系统时，文件组织结构完整保留
- **跨设备迁移**：从旧路由器迁移到新服务器，无需从头开始
- **配置可移植**：轻松导出导入您的设置

### ⚡ 自适应性能

FileUni 自动适应您的硬件环境：

- **低端设备**（32-64MB 内存）：核心文件服务功能可靠运行
- **中端设备**（128MB-1GB 内存）：流畅的网页界面、多用户并发、后台任务
- **高端设备**（2GB 以上内存）：完整功能集，包括缓存、索引和并行处理

> **提示**：更多资源意味着更好的性能，但 FileUni 永远不会阻止您在有限硬件上运行。

## 使用场景示例

### 📸 照片备份
把手机里的照片自动备份到家里的设备，释放手机空间，同时保留珍贵回忆。

### 🎬 家庭影院
将电影、音乐存储在 FileUni 上，全家在任何设备上都能观看。

### 📂 工作协作
小团队共享项目文件，每个人都能看到最新版本，告别文件传来传去。

### 💾 资料归档
将不常用但又不想删除的文件归档存储，需要时随时调取。
