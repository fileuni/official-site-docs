# 快速开始指南

本指南将帮助您在几分钟内搭建起 FileUni 文件服务器。

## 安装方式

### 方式一：命令行一键安装（推荐）

在终端中运行以下命令：

```bash
curl -fsSL https://fileuni.com/install.sh | sh
```

安装完成后，系统会自动启动服务。

### 方式二：下载预编译版本

1. 访问 [下载页面](https://fileuni.com/download)
2. 根据您的设备选择对应版本：
   - Windows：下载 `.exe` 文件
   - Mac：下载 `.dmg` 文件
   - Linux：下载对应架构的二进制文件
   - 路由器/ARM 设备：选择 ARM 版本
3. 解压后运行程序

### 方式三：Docker 部署

如果您熟悉 Docker，可以使用以下命令：

```bash
docker run -d \
  --name FileUni \
  -p 19000:19000 \
  -v /your/data/path:/data \
  fileuni/FileUni:latest
```

## 首次配置向导

首次启动时，系统会引导您完成简单配置：

### 第 1 步：选择存储位置

选择您想要用于存储文件的文件夹：
- 可以是内置硬盘
- 也可以是外接 USB 硬盘
- 建议选择空间充足的磁盘

### 第 2 步：创建管理员账户

设置您的管理员账号和密码：
- 用户名：建议使用容易记住的名称（不是邮箱）
- 密码：建议设置 8 位以上，包含字母和数字

> 💡 **提示**：这是最高权限账户，请妥善保管密码

### 第 3 步：选择运行模式

根据您的设备性能选择：

| 模式 | 适合设备 | 特点 |
|------|---------|------|
| **省电模式** | 路由器、旧手机 | 限制并发，降低资源占用 |
| **标准模式** | 普通电脑 | 平衡性能与资源使用 |
| **性能模式** | 专用服务器 | 充分发挥硬件性能 |

## 访问您的服务器

配置完成后，您可以通过以下方式访问：

### 网页访问
打开浏览器，输入：
```
http://localhost:19000/ui
```

如果是在其他设备上访问，将 `localhost` 替换为服务器 IP 地址。

### 手机 APP 访问
FileUni 支持标准协议，您可以使用以下 APP：
- **Android**：Solid Explorer、MiXplorer
- **iOS**：Documents、FileBrowser

连接时选择 WebDAV 或 SFTP 协议，输入服务器地址和账号即可。（注：FTP 仅支持明文传输，建议使用 SFTP）

## 基本操作

### 上传文件
1. 登录网页界面
2. 点击"上传"按钮
3. 选择要上传的文件或文件夹
4. 等待上传完成

### 创建文件夹
1. 点击"新建文件夹"按钮
2. 输入文件夹名称
3. 点击确认

### 分享文件
1. 选中要分享的文件或文件夹
2. 点击"分享"按钮
3. 设置分享密码和有效期
4. 复制分享链接发送给他人

## 常见问题

**Q: 忘记管理员密码怎么办？**
A: 在服务器上运行命令：`FileUni -a newpassword` 来重置密码

**Q: 如何在外网访问？**
A: 需要配置内网穿透或 DDNS，具体请参考[远程访问指南](./remote-access)

**Q: 如何备份数据？**
A: 建议定期备份存储文件夹，或使用系统的导出功能
